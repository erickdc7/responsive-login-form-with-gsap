# Responsive Login Form With GSAP

This project is part of a YouTube tutorial in which we build a responsive login form from scratch using HTML, CSS, JavaScript & GSAP.

## [Watch it on YouTube](https://youtu.be/Dis15uI1DSk)

## Project Features
- Fully responsive login form (Mobile First Methodology).
- Clear and semantic HTML structure.
- Use of CSS variables.
- Smooth animations with GSAP.
  
## Download the source code directly
1. Click the green **code** button.
2. Click **Download ZIP**.
3. Extract the ZIP file and open the project in your code editor.
